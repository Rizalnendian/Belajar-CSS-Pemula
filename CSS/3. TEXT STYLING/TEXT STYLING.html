color

Berfungsi untuk memberi nama pada tulisan, value dari atribut ini bisa dengan nama warna, kode hex dan kode rgb

text-align

Berfungsi untuk mengatur format paragraf/text, value dari atribut ini yaitu left, right, center, justify

text-indent

Berfungsi untuk memberi indentasi paragraf/text dengan satuan hitung px dan %

text-decoration

Berfungsi untuk mengatur dekorasi paragraf/text seperti none (untuk menghilangkan dekorasi), underline, overline dan line-throught

text-transform

Berfungsi untuk mengubah jenis huruf menjadi huruf besar, kecil/kapital pada paragraf/text valuenya yaitu (untuk menghilangkan efek), lowercase, uppercase, capitalize

letter-spacing

Berfungsi untuk mengatur spasi antar huruf dengan satuan hitung px dan normal (untuk menormalkan kembali)

word-spacing

Berfungsi untuk mengatur spasi antar kata dengan satuan hitung px dan normal (untuk menormalkan kembali)
