JENIS JENIS FONT STYLING

font-family
mengatur jenis font yang akan digunakan
font-size
mengatur ukuran font
font-weight
mengatur ketebalan font
font-variant
mengubah font menjadi small caps
font-style
mengubah font menjadi cetak miring
line-height
mengatur spasi antar baris

Anatomi Atribut font-family

font family memiliki value nama font dan generic family, contoh
font family: arial, verdana, sans-serif;
arial dan verdana adalah nama font, dan sans-serif adalah nama generic family

font-size

font-size memiliki satuan hitung seperti px, %, em, dan beberapa value yang sudah disediakan

font-weight

font-weight memiliki value seperti lighter, normal, 100-900, bold, border dan masih banyak lagi

font-variant

font-variant memiliki value seperti normal, small-capss dan masih banyak lagi

Contoh Value Normal
Contoh Value Small-Caps

font-style

font-style memiliki value seperti normal, italic, oblique dan masih banyak lagi

italic berlaku hanya untuk font yang memiliki font italic, bagi font yang tidak memiliki style italic maka bisa menggunakan oblique yang akan memiringkan font tersebut

line-height

line-height memiliki value seperti normal, px, em dan masih banyak lagi

shorthand font yaitu dengan mengetik misal body{font: italic normal bold 16px/18px helvetica,arial, sans-serif}
jadi urutann6ya adalah font-style, font-variant, font-weight, font-size(wajib), line-height dan font-family(wajib)
